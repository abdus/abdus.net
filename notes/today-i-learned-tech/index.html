ES6: Private method using Symbol()

Symbol in JavaScript(ES6) is a data type. The speciality about Symbols are that the value is unique. Using this feature, one could abstract methods of ES6 class. Here’s an example:

(() => {
    const FORMAT_MSG = Symbol();
    class SayHello {
        constructor(name) {
            this[FORMAT_MSG](name);
        }

        [FORMAT_MSG](name) {
            this.message = `Hello, ${name}`
        }

        done() {
            alert(this.message);
        }
    }

    window.SayHello = SayHello;      // make it available to global scope
})()
